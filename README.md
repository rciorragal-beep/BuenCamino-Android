## Buen Camino - Aplicación para Android 

Este repositorio incluye el apartado del cliente de la aplicación Buen Camino, creada como Proyecto Final del Ciclo.
Buen Camino es una app para Android diseñada para ofrecer un acceso sencillo a información y recursos sobre el embarazo, el postparto, la lactancia y los cuidados iniciales del bebé.

## Tecnologías empleadas

- Android Studio
- Kotlin
- XML
- Retrofit
- SDK de Google Maps para Android


## Detalles principales

La aplicación ofrece la posibilidad de explorar diversas categorías de recursos, ver los elementos relacionados con cada una de ellas y acceder a enlaces externos directamente desde la app.
Asimismo, incorpora una sección de mapas que permite visualizar los recursos disponibles según su ubicación, ej. aquellos vinculados a matronas.

## Conexión con el backend

La aplicación utiliza una API REST que ha sido creada con Node.js y Express.  
El backend se encuentra conectado a una base de datos PostgreSQL que está ubicada en Neon y desplegada en Railway.
